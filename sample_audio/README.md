# 示例音频文件夹

此文件夹用于存放演示和测试用的音频样本。

## 📁 文件结构

```
sample_audio/
├── README.md                    # 本文件
├── voice_samples/               # 音色克隆样本
│   ├── grandmother_30s.mp3     # 奶奶的声音（30秒）
│   ├── grandfather_30s.mp3     # 爷爷的声音（30秒）
│   └── mother_30s.mp3          # 妈妈的声音（30秒）
└── test_inputs/                 # 测试用输入
    ├── greeting.wav            # 问候语
    ├── concern.wav             # 关心的话
    └── sharing.wav             # 分享近况
```

## 🎙️ 音频要求

### 用于音色克隆的样本：

- **时长：** 30-60 秒
- **格式：** MP3 或 WAV
- **采样率：** 16kHz 或更高
- **音质：** 清晰、无杂音、无背景音乐
- **内容：** 自然对话，包含多种语调和情绪
- **音量：** 适中，不要过大或过小

### 用于测试对话的输入：

- **时长：** 3-10 秒
- **格式：** WAV（推荐）或 MP3
- **采样率：** 16kHz
- **内容：** 一句完整的话

## 📝 录音建议

### 1. 环境准备
- 安静的房间
- 关闭风扇、空调等噪音源
- 使用手机或专业麦克风

### 2. 录制内容示例

**适合克隆的内容（30秒）：**
> "乖囡啊，最近过得怎么样？天气冷了，要多穿点衣服，别着凉了。吃饭要按时吃，不要总是吃外卖。工作虽然重要，但是身体更重要。记得常回家看看，奶奶想你了。"

**测试对话的内容：**
- "奶奶，我今天好累啊"
- "我在新加坡参加比赛"
- "你最近身体怎么样？"
- "我好想你啊"

### 3. 录音工具推荐

**移动端：**
- iOS：Voice Memos（自带）
- Android：录音机（自带）

**桌面端：**
- Mac：QuickTime Player
- Windows：录音机 / Audacity
- Linux：Audacity

**在线工具：**
- https://online-voice-recorder.com/

## 🔧 音频转换

如果需要转换格式或调整采样率：

```bash
# 使用 ffmpeg 转换
ffmpeg -i input.m4a -ar 16000 output.wav

# 批量转换
for f in *.m4a; do 
    ffmpeg -i "$f" -ar 16000 "${f%.m4a}.wav"
done
```

## ⚠️ 注意事项

1. **隐私保护：** 不要上传包含敏感信息的录音
2. **版权：** 确保有权使用录音中的声音
3. **文件大小：** 单个文件不超过 10MB
4. **命名规范：** 使用英文和数字，避免特殊字符

## 📥 获取样本

如果没有录音设备，可以：

1. 使用文字转语音工具生成测试样本
2. 从免费音频素材网站下载（注意版权）
3. 使用项目提供的示例（如果有）

## 🎯 快速测试

创建一个简单的测试样本：

```bash
# 使用系统 TTS 生成测试音频（Mac）
say -o test_voice.aiff "Hello, this is a test voice sample"
ffmpeg -i test_voice.aiff -ar 16000 test_voice.wav
```
